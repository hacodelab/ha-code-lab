🚀 Jinsi ya Ku-Upload GitHub

Hatua 1: Tengeneza GitHub Account

  1. Nenda github.com
  2. Click Sign up
  3. Jaza taarifa zako (email, username, password)
  4. Confirm email yako

Hatua 2: Tengeneza Repository Mpya

  1. Ingia kwenye GitHub account yako
  2. Click ”+” icon juu kulia → New repository
  3. Jaza taarifa:
    • Repository name: ha-code-lab (au jina lolote)
    • Description: “HA Code Lab - Programming Education Website”
    • Public (ili watu waione)
    • ☑️ Check “Add a README file”
  4. Click “Create repository”

Hatua 3: Upload Files

Njia 1: Direct Upload (Rahisi Zaidi)

  1. Kwenye repository yako, click “Add file” → “Upload files”
  2. Drag & drop FAILI ZOTE 8:
    • index.html
    • style.css
    • script.js
    • Python_Programming_Notes.pdf
    • C_Programming_Notes.pdf
    • HTML_Course_Notes.pdf
    • Java_Programming_Notes.pdf
    • CPP_Programming_Notes.pdf
  3. Andika commit message: “Initial commit - Added website files”
  4. Click “Commit changes”

Njia 2: Git Command Line (Advanced)

# Install Git kwanza (ikiwa huna)
# Windows: Download from git-scm.com
# Linux: sudo apt install git
# Mac: brew install git

# Clone repository
git clone https://github.com/USERNAME/ha-code-lab.git
cd ha-code-lab

# Copy faili zako zote kwenye folder hii
# Kisha:
git add .
git commit -m "Initial commit - Added website files"
git push origin main

Hatua 4: Activate GitHub Pages

  1. Kwenye repository yako, click “Settings”
  2. Sidebar, click “Pages”
  3. Under “Source”, chagua “main” branch
  4. Click “Save”
  5. Subiri dakika 2-3
  6. Website yako itakuwa live kwenye: 
    https://USERNAME.github.io/ha-code-lab/
